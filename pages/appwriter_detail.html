AppWriter

Project Information

AppWriter provides help with reading and writing through a range of features such as context-based word suggestions, text-to-speech, OCR scanning and a pdf reader.

  • Role: Developer
  • Responsibility: Development, Bug Fixes, Testing and Release
  • Tool: Xcode (10.1)
  • Programming Language: Objective-C
  • Databse: SQL, CoreData, UserDefaults
  • API: REST
  • Notifications: APNS
  • Functionality: Custom Keyboard, OCR, Text To Speech, Speech To Text, Create Document, Google Drive SDK, OneDrive SDK, Highlight Speech, Localisation
