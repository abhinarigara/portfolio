TxtAnalyser

Project Information

TxtAnalyser is a unique language application that finds and corrects spelling and grammatical mistakes.It also supports multiple languages and provides suggested video. Suggested videos come from YouTube which users can play inside an iOS application.

  • Role: Developer
  • Responsibility: Development, Bug Fixes, Testing and Release
  • Tool: Xcode (10.0)
  • Programming Language: Objective-C
  • Architecture: MVC
  • Databse: SQL, CoreData, UserDefaults
  • API: REST
  • Functionality: YouTube Video, Core Text, Analyze grammar, Analyze Spellings, Localisation
