CyVerse

Project Information

Using CyVerse, the organization user can communicate with anyone in his organization on text messages, Audio calls, Video calls and conversation in secure chat rooms. Users can have intra business and inter business communication with the unique feature of CyVerse Federation with end-to-end encryption.

  • Role: Sr.Developer
  • Responsibility: Development, Bug Fixes, Testing, Documentation and Release
  • Tool: Xcode (14.1)
  • Programming Language: Swift (5.7.1), Objective-C
  • Architecture: MVVM, Co-ordinator
  • Databse: Matrix, SQL
  • API: REST
  • Notifications: APNS
  • Functionality: Realtime Chat, VOIP Calls, Push Notifications, Group Calls, Noticeboard posting
